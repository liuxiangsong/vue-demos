# Markdown 语法

## 标题（1~6 级标题）

- # 一级标题

- ## 二级标题

## 字体

- 斜体：
  - _斜体文本_
  - _斜体文本_
- 粗体
  - **粗体文本**
  - _粗体文本_
- 粗斜体
  - **_粗斜体文本_**
  - **_粗斜体文本_**
- 下划线
  - <u>下划线</u>
- 删除线
  - ~~删除线~~

## 区块

> 区块一
>
> > 区块二

## 无序列表

- 选项 1

* 选项 2

- 选项 3

## 有序列表

1. 选项 1
2. 选项 2
3. 选项 3

## 分割线

---

---

## 代码框

`var name='Sam';`

```javascript
var name = 'Sam'
```
